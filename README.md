# NCERT Books Download Website

This project is a simple and clean website where students can download NCERT Books for all classes.  
The aim of this website is to provide quick access to NCERT PDFs so that students don't have to search everywhere.

## Features
- Download NCERT books for multiple classes
- Clean and easy-to-use interface
- Fast loading pages
- Direct PDF download links
- Mobile friendly UI

## How to Use
1. Open the website in your browser.
2. Choose your class (e.g., Class 6, Class 7, Class 10, Class 12, etc.).
3. Select the subject.
4. Click on the chapter/book to download the PDF.

## Technologies Used
- HTML
- CSS
- JavaScript (optional) #not use this language
- PDF links

## Purpose
This website helps students easily download NCERT books for study and revision.
